UT4 Installer
UT4 Installer - v1.0.3
  • Download link

Download, unzip, run UT4_Installer.exe, and follow installer instructions for a simple UT4 installation without epic games launcher.

Click here to watch a demo video

Installer features:

  • Preconnected to new master server
  • Preloaded with UT4UU v10.1.6
  • Checks/installs game dependencies normally installed by epic games launcher
  • Creates desktop shortcut with necessary parameters to start game without epic launcher
  • Includes MonkeyPatch v1.0 and Audio Fix v1.1
